# Advisor Check-In

This skill provides detailed information from your academic framework when you need it.

Your main CLAUDE.md gives you core identity and current priorities. This skill provides deeper detail without overwhelming your everyday context.

## What This Skill Provides

**Current Priorities**: Detailed weekly/phase task breakdowns
**Methodology**: How to create skills, use lab techs, conduct reflection, structure documentation
**Assessment**: Competency rubrics for self-evaluation
**Context**: Long-term vision, academic progression details
**Strategic Guidance**: When to consult advisor vs. decide independently

## How to Use

Be specific about what you need:

Good: "Starting CLAUDE.md conversion, need methodology for creating skills"
Good: "Time for monthly review, need competency self-assessment guide"
Good: "Feeling unclear on direction, need long-term vision context"
