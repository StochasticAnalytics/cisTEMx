## What is cisTEMx?

cisTEMx is a GPU-accelerated fork of cisTEM (computational imaging system for transmission electron microscopy) designed for high-performance single-particle analysis and cryo-EM image processing.

### Key Features

- **GPU Acceleration** - CUDA-optimized algorithms for faster processing
- **Modern Architecture** - Refactored codebase with improved maintainability
- **Flexible Workflows** - Support for various cryo-EM processing pipelines
- **Cross-Platform** - Runs on Linux, macOS, and Windows with WSL
